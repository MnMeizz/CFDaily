# CF Daily — Codeforces 每日一题插件

一个轻量级的 AstrBot 插件，每天为用户随机推荐一道 Codeforces 题目，并生成题面图片。

## ✨ 功能

- **每日一题**：从 Codeforces 题库中随机抽取一道题目，通过图片呈现完整的题面描述、输入/输出格式、样例和标签。

## 🛠️ 安装方法

### 方式一：通过 Astrbot 插件商店安装（推荐）
1. 进入 Astrbot 的管理面板。
2. 在 **插件** 页面搜索 `CFDaily` 或通过仓库 URL 安装。
3. 点击 **安装** 并等待加载完成。

### 方式二：手动安装
将本项目克隆到 Astrbot 的 `plugins` 目录下：
```bash
cd Astrbot/data/plugins
git clone https://github.com/MnMeizz/CFDaily.git
```
## 📖 使用方法

插件安装成功并启用后，您可以在聊天窗口发送以下指令进行交互。

### 🎮 核心指令

| 指令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `每日一题` | 推荐每日一题。 | `/每日一题` |
